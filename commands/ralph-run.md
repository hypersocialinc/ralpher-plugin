# Ralph Run

This command is an alias for `/ralph-go`. Run that command instead.

## Usage

```
/ralph-go
```

The `/ralph-go` command provides flexible execution modes:
- **Terminal Script Mode** - Runs in your terminal, can background
- **Autonomous Session Mode** - Runs in Claude Code with real-time visibility

See `/ralph-help` for full documentation.
